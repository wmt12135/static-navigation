# 超简单的可响应静态导航网站

## 网址：https://wmt12135.github.io/

### 若想更改网页中导航的网址，只需修改 ./src/views/play/play.json 与 ./src/views/study/study.json 中的数据即可。
<br><br>

#### 初始化项目
``` shell
$ git clone https://github.com/wmt12135/static-navigation.git
$ cd static-navigation
$ npm install
```
#### 预览测试
``` shell
$ npm run start
```
#### 构建打包
``` shell
$ npm run build
```

### 然后使用 github pages 构建你的网站或者在你自己构建的服务器下，将 ./dist 文件夹下的文件复制到你的项目根目录中，即可构建你自己的导航网站了。
